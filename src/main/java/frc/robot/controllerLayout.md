
# Controller Layout for single-controller compBot

- **Joysticks:** Swerve Steering
- **Left Trigger:** Boost
- **Left Bumper:** Field oriented
- **Y:** Reset Gyro
- **Right Trigger:** Shoot
- **Right Bumper:** Flush (Reverse shooter, indexer, and intake)
- **X:** Toggle Intake Deployment
- **A:** Run Intake
- **B:** Run Indexer
- **POV DOWN:** Set Hood to Zero
- **POV UP:** Set hood to low height
- **POV RIGHT:** Set Hood to Medium
- **POV LEFT:** Set Hood to Highest
